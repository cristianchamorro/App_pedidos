# Logo Assets

## Required Logo Files

Place your restaurant logo files in this directory with the following names:

- `logo.png` - Main logo image (recommended size: 512x512px or larger)
- `logo_white.png` - White version of logo for dark backgrounds (optional)

## Recommended Specifications

### Logo Image
- **Format:** PNG with transparent background
- **Size:** 512x512px minimum (can be larger, will be scaled)
- **Color:** Should match the app theme colors:
  - Primary Red: #D32F2F
  - Orange: #FFA726
  - Golden: #FFB300

### Usage
The logo will be displayed:
- Login choice page (welcome screen)
- Admin login page
- App bar (optional, can be added later)

## Placeholder

If no logo is provided, the app will display a default restaurant icon.
To use your custom logo:
1. Add your logo file as `logo.png` in this directory
2. Make sure it's referenced in `pubspec.yaml` under assets
3. Rebuild the app

## Color Palette Reference

The app uses a warm color scheme suitable for food businesses:
- **Primary (Red):** #D32F2F
- **Primary Light:** #EF5350
- **Primary Dark:** #C62828
- **Secondary (Orange):** #FFA726
- **Accent (Golden):** #FFB300
- **Success (Green):** #4CAF50
